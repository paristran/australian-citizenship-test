# 🇦🇺 Australian Citizenship Practice Test

Prepare for your Australian citizenship test with 100+ practice questions based on "Our Common Bond" — the official test resource.

## ✨ Features

- **100+ Practice Questions** — Comprehensive coverage of all testable content
- **Test Mode** — 20 questions, 45 minutes, 75% to pass (just like the real test)
- **Study Mode** — Browse all questions by category with explanations
- **Progress Tracking** — See your score in real-time
- **Beautiful UI** — Clean, Apple-inspired design

## 📚 Topics Covered

- Australia and its people
- Democratic beliefs, rights and liberties
- Government and the law
- Australian values

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## 🧪 About the Test

The Australian citizenship test:
- 20 multiple choice questions
- 45 minutes to complete
- Need 75% (15/20) to pass
- Based on "Our Common Bond" booklet
